Anatomy of an HTML Tag

In the snippet, there is a paragraph tag and an anchor tag. For more information:

  1. P tags
  2. Anchor tags
The anchor tag is a child of the paragraph tag. <p> is an opening tag while </p> is a closing tag.

Anchor Attributes

  1. Attribute: href, Value: "https://google.com". This is a link to content
  2. Attribute: id, Value: "the-best-link". This is a unique identifier of this element
  3. Attribute: class, Value: "banana" This is used to apply CSS
  4. Attribute: target, Value: "_blank" This indicates where to open the link
